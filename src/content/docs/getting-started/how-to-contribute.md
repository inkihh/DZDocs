This site only works if people write things down. The good news: if you can write
Markdown, you can contribute.

## The short version

1. Find a page that's wrong, thin, or a stub — or a topic that's missing entirely.
2. Use the **Edit** link at the bottom of any page (or open the file in the repo).
3. Make your change in Markdown and open a pull request.

That's it. You don't need repo access to contribute — pull requests are the path for
everyone, and core contributors review and merge them.

:::tip[Not sure where to start?]
Stub pages (flagged with a **stub** badge in the sidebar) already have an outline
waiting to be filled in. Picking one of those up is the easiest way in.
:::
